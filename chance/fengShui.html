進金：

金罐準備：（蓮花金罐）

1.畫好中線（基準線）

2.在金罐上刻名字（用貼紙也可以），貼在對準中線的位置，長度要符合生老病死苦的生或老（六個字“生”或七個字“老”）

Ex: 第二十世秉勳。 第十九世偉弘。十八世祖振旺。 十八玉女朱氏。

3-1.有骨灰的先將骨灰放入金罐內，填入3寸粗沙（代表金沙的意思）

3-2.沒骨灰的準備紅磚（紅磚要洗淨曬乾）寫名字放入（同金罐上的字），填入3存金沙

4.灑淨，用做好的淨水噴灑在金罐內即完成灑淨動作

金井準備：

1.金井內放入新的細沙(金沙)，高度以金罐放入後離金井口一寸為準（金井封蓋時才不會壓到金罐）

2.金井暖金，壽金、刈金、福金各一張，用夾子夾著，點燃後在金井內畫圓直到燒完即完成暖金動作

羅盤準備：

Ex:卯山辛卯(座東朝西)，將羅盤上卯山辛卯的位置對準白線，轉動羅盤將指針與內盤的紅線重疊，白線位置則為東西向


進金流程

清理金井內舊的細沙 => 將新的細沙填入金井內(金罐放入後離金井口一寸) => 金井暖金 => 放入金罐 => 校對方向(羅盤要先校正好，羅盤的白線與金罐的中線要為一直線) => 金罐灑淨 => 金罐封蓋 => 金井封蓋

進金要按照順序，事先要先去問過恩主公
